配置方式（任选其一）

方式 1：Access Token（与 mcp.json 相同）

适合快速测试，支持「前往 PayPal 付款」链接流程。

# PowerShell — 从 mcp.json 导入
.\setup-paypal.ps1 -FromMcp

# 或手动设置
[Environment]::SetEnvironmentVariable("PAYPAL_ACCESS_TOKEN", "你的Token", "User")
[Environment]::SetEnvironmentVariable("PAYPAL_ENVIRONMENT", "SANDBOX", "User")

方式 2：Client ID + Secret（推荐，支持 PayPal 按钮）

在 PayPal Developer 创建 Sandbox App 获取。

.\setup-paypal.ps1 `
  -ClientId "你的Client-ID" `
  -ClientSecret "你的Secret" `
  -Environment "sandbox"

方式 3：项目 .env 文件

复制 .env.example 为 .env 并填写，重启服务即可。

重启服务

cd "项目目录"
npm start
